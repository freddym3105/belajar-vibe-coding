# Belajar Vibe Coding — Scaffold

High-level scaffold for a Bun + Elysia + Drizzle + MySQL project. This repository contains minimal files to guide implementers — install dependencies and complete configuration before running.

Quick start (for implementer):

```bash
# install (using Bun)
bun install

# run dev server
bun dev
```

See `issue.md` for the original planning details.
